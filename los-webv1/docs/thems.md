| Field Name           | Purpose / Usage in UI                                  | Example Usage                          |
|----------------------|---------------------------------------------------------|----------------------------------------|
| `primaryColor`       | Main branding color (buttons, links, highlights)        | `bg-primary`, `text-primary`, borders  |
| `secondaryColor`     | Accent or complementary color (badges, secondary CTAs)  | `bg-secondary`, `text-secondary`       |
| `backgroundColor`    | Main page background                                    | `bg-background`, body background       |
| `surfaceColor`       | Card or modal background                                | `bg-surface` for `div`, `card`         |
| `primaryTextColor`   | Default or main text color                              | `text-textPrimary`                     |
| `secondaryTextColor` | Subtle/secondary text (placeholders, captions)          | `text-textSecondary`                   |
| `successColor`       | Success states (alerts, icons, buttons)                 | `text-success`, `bg-success`           |
| `warningColor`       | Warning states (tooltips, icons, banners)               | `text-warning`, `bg-warning`           |
| `errorColor`         | Error states (input borders, alert messages)            | `text-error`, `bg-error`               |
